# "티"를 빼야 건강식품이 산다 — AI 감리자의 네거티브 키워드 함정

## 소재 요약

기타가공품 75K를 건강식품으로 재태깅하는 과정에서, 감리자(Gemini)가 제안한 네거티브 키워드 "티"가 362건의 정상 건강식품(뷰티콜라겐, 액티브유산균, 플래티넘 등)을 오살하는 문제를 발견. "아이스티"만 생각하고 넣은 키워드가 한국어 외래어 표기 특성(뷰**티**, 액**티**브)과 충돌. 데이터 실증으로 90:1 오살 비율을 증명하여 감리자가 "완벽하게 패배" 인정.

## Story 관점 (운영 블로그)

- **제목안**: "AI가 AI를 감리할 때 — 데이터가 이기는 논쟁법"
- **핵심**: 논리적으로 완벽해 보이는 제안도 실제 데이터 분포 앞에서 무너질 수 있다
- **구조**:
  1. 감리자의 합리적 제안 ("단백질 쿠키는 간식이지 건강식품이 아니다")
  2. 구현자의 반론 준비 — 663건 교차 분석
  3. "티" 367건 중 362건이 건강식품이었다 (뷰티, 액티브, 플래티넘...)
  4. 감리자의 "완벽한 패배" 인정
  5. 교훈: 일반론 vs 컨텍스트 (기타가공품이라는 1차 필터의 마법)

## Tech 관점 (기술 블로그)

- **제목안**: "NLP 없이 75K 제품 분류하기 — 키워드 + 제형 네거티브 전략"
- **구조**:
  1. 문제 정의: product_type_text='기타가공품' 75K를 건강식품으로 선별
  2. 3-Tier 키워드 설계 (명확/전통/넓은) + 환 접미사 Regex
  3. 네거티브 키워드 설계 — "제형 기반만" 원칙
  4. 한국어 외래어의 함정: 뷰티/액티브/멀티/플래티넘에 숨은 "티"
  5. 결과: 14,668건 재태깅, dry-run 100건 샘플 오분류 0건

## 5대 인사이트

1. **컨텍스트가 키워드를 이긴다** — "석류즙은 음료"가 맞지만, 기타가공품 안의 석류는 99% 건강식품
2. **한국어 외래어 함정** — "티" 차단 시 뷰티/액티브/멀티 등 362건 오살 (90:1 비율)
3. **제형 기반 네거티브** — 젤리/쿠키/캔디 등 형태가 확실한 것만 안전
4. **AI 감리 ≠ 무조건 수용** — 데이터 실증으로 반론하면 감리자도 수긍
5. **display_category 아키텍처** — 원본(product_type_text) 보존 + 표시용 분리가 안전한 재태깅의 전제
