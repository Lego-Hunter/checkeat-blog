## 이 연재물에 대해

CheckEat(첵잇)은 식품 성분을 쉽게 이해하고 비교할 수 있게 도와주는 웹 서비스입니다.
이 블로그는 그 서비스를 만들어가는 과정을 두 가지 관점에서 기록합니다.

- **운영 블로그**: 왜 이런 선택을 했는지, 무엇이 어려웠는지 — 개발 지식 없이도 읽을 수 있습니다.
- **기술 블로그**: 어떻게 구현했는지 — 개발자를 위한 기술적 기록입니다.
